# SmartConnect: Advanced School Management System

## 🎯 Aim
The aim of **SmartConnect** is to bridge the communication gap between educational institutions, teachers, students, and parents. It provides a centralized, secure, and highly interactive platform to manage all school-related activities—from administrative tasks to academic progress—using a modern, mobile-first approach.

---

## 🚀 Core Features

### 👤 Multi-Role Architecture
The application is built on a robust multi-role system, ensuring that each user has access only to relevant information and tools.

#### 🛠️ Admin Tab
*   **User Management:** Full control over managing Principal, Teachers, Students, and Parents.
*   **Fees Management:** Create and monitor fee structures, track payments, and manage installments.
*   **E-commerce Hub:** Manage school-related products (uniforms, books, etc.) and orders.
*   **Feedback & Communication:** Centralized view of all feedback and contact messages from various roles.

#### 🏫 Principal Tab
*   **Academic Oversight:** View and manage timetables, exams, and notices.
*   **Teacher Collaboration:** Monitor and communicate with the teaching staff.
*   **Financial Tracking:** Overview of fee collections and institutional payments.

#### 👨‍🏫 Teacher Tab
*   **Class Management:** Mark and manage student attendance, homework, and assignments.
*   **Academic Tools:** Upload syllabi, manage exam updates, and post important notices.
*   **Student Monitoring:** Direct access to student lists and their academic performance.

#### 🎓 Student Tab
*   **Personal Dashboard:** Real-time access to attendance, marks, and timetable.
*   **Interactive Learning:** View and download assignments, syllabi, and homework.
*   **E-commerce:** Browse and purchase school supplies directly from the app.
*   **Administrative Access:** Track fee payments and contact administration for queries.

#### 👨‍👩‍👧‍👦 Parent Tab
*   **Child Monitoring:** Track attendance, exam results, and academic progress of their children.
*   **Fee Management:** View fee structures and pay fees directly through the app.
*   **Stay Informed:** Receive notices, timetables, and school updates instantly.
*   **Direct Feedback:** Share thoughts or concerns directly with the school administration.

## ✨ Dynamic Animations
SmartConnect is imbued with a professional UI/UX, powered by a custom `UIAnimator` suite to provide a smooth and engaging user experience.

*   **Pulse Animation:** Continuous breathing effect for core brand icons and logos.
*   **Slide & Fade:** Smooth entry animations for text views and edit texts (Slide from right/up).
*   **Scale & Overshoot:** Bouncy scale-up effect for buttons and dashboard cards.
*   **Interactive Feedback:** Instant ripple and scale-down effects on button clicks.
*   **Staggered Loading:** Sequential entry animations for list items and linear layout contents.
*   **Looping Pulse:** Continuous "Sending..." pulse effect during OTP and authentication processes.

---

## 🛠️ Technology Stack
*   **Platform:** Android (Java)
*   **Database:** Firebase Realtime Database (for instant sync)
*   **Authentication:** Firebase Auth (with OTP-based verification)
*   **UI Components:** Material Design 3 (MaterialCardView, MaterialButton, etc.)
*   **Image Processing:** Base64 encoding for profile photos.
*   **Messaging:** Integrated Gmail service for secure OTP delivery.

## 📥 Installation
1.  Clone the repository: `git clone https://github.com/yourusername/smartconnect.git`
2.  Open the project in **Android Studio**.
3.  Connect your **Firebase project** and download the `google-services.json` file.
4.  Build and run the app on your emulator or physical device.
